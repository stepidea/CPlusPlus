# 创建第一个 Qt 程序

    ## 点击创建项目后，选择项目路径（不能有中文）以及给项目起名称（名称不能有中文、空格）
    ## 默认创建有窗口类，myWidget，基类有三种选择：QWidget、QMainWindow、QDialog
    ## main 函数
        ### QApplication 应用程序对象，有且只有一个
        ### myWidget w 实例化窗口对象
        ### w.show() 调用show函数，显示窗口
        ### return a.exec() 让应用程序对象进入到消息循环机制中，代码阻塞到当前行
